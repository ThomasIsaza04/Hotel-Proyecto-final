# React + TypeScript — Control Flow + Operaciones Aritméticas (Vite)

## Scripts
- `npm i`
- `npm run dev` — desarrollo (Vite)
- `npm run build` — build (dist/)
- `npm run preview` — previsualizar dist/

## Qué incluye
- Variables y operaciones (`VariablesOps.tsx`)
- Decisiones (`DecisionDemo.tsx`)
- Ciclos (`LoopsDemo.tsx`)
- **Aritmética aplicada** (`ArithmeticOps.tsx`) con utilidades en `src/lib/math.ts`
